Invest Local

Invest Local was my final project at The Iron Yard

Invest Local is a website that gives consumers tools to evaluate banks based on a set of criteria. For example, local commitment, location, size, financial strength, lending data and number of complaints. The data is presented in an intuitive manner that allows for comparison banks.

You can visit the website to view our hardwork. Invest-Local.herokuapp.com

Medieval Game

A text based choose your own adventure style game made in HTML, CSS, and Javascript.

Medieval Game is a text based game that I created during my time at The Iron Yard. It started out very simple with only pictures and words. With Javascript I have added several variables that carry over from page to page and plan to add a few mini games on some of the pages. This game is still a work in progress.

You can preview the game here.

Road Rage

Road Rage is a project to simulate the behavior of drivers in traffic on a new road in order to determine the optimal speed limits.

This is a homework assignment that I completed with a partner at The Iron Yard. It simulates cars driving down a straight road trying to achieve the maximum speed they can get without crashing into the car in front of them.
